# Installation

Get your first FormKitCN block installed and running in under a minute.

## Prerequisites

Your project needs:

- Next.js 13+ (App Router)
- React 18+
- TypeScript
- Tailwind CSS
- shadcn/ui initialized — run `npx shadcn@latest init` if you haven't already

<Callout variant="info">
  FormKitCN components are shadcn registry blocks. The shadcn CLI handles all dependency installation automatically.
</Callout>

## Install your first block

Pick a block and add it to your project with one command. Here's the most common starting point — the dynamic form field:

<InstallCmd file="form-field.json" />

The CLI will copy the component files into your project and install all required npm packages.

## Sonner setup

Blocks that depend on `sonner` also require a toaster in your root layout. This is currently needed for:

- `form-dynamic-template.json`
- `multistep-form-template.json`
- `form-template-01.json`

Add this once in `app/layout.tsx`:

```tsx
import { Toaster } from "@/components/ui/sonner"

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        {children}
        <Toaster position="top-center" />
      </body>
    </html>
  )
}
```

## Quick example

After installing `form-field`, here's a minimal working form using `DynamicFormField` with `react-hook-form` and Zod:

```tsx
import { useForm } from "react-hook-form"
import { zodResolver } from "@hookform/resolvers/zod"
import { z } from "zod"
import { Form } from "@/components/ui/form"
import { Button } from "@/components/ui/button"
import { DynamicFormField } from "@/components/forms/form-field/DynamicFormField"
import { FormFieldType } from "@/components/forms/form-field/constants"

const schema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
})

export function LoginForm() {
  const form = useForm({
    resolver: zodResolver(schema),
    defaultValues: { email: "", password: "" },
  })

  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit(console.log)} className="space-y-4">
        <DynamicFormField
          control={form.control}
          name="email"
          fieldType={FormFieldType.INPUT}
          label="Email"
          placeholder="you@example.com"
          required
        />
        <DynamicFormField
          control={form.control}
          name="password"
          fieldType={FormFieldType.PASSWORD}
          label="Password"
          required
        />
        <Button type="submit">Sign in</Button>
      </form>
    </Form>
  )
}
```

<Callout variant="tip">
  Want to skip the react-hook-form boilerplate? Use **FormBuilderStandard** instead — it handles schema, layout, and submission automatically. See the [FormBuilderStandard docs](/docs/form-builder).
</Callout>
