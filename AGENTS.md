# PROJECT CONTEXT

FormKitCN is a **developer toolkit and component registry system** built using:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zod (schema validation)
- React Hook Form
- Redux (for persisted client state)

---

# 🚀 CORE PURPOSE

FormKitCN enables developers to:

- Build dynamic forms using schema-driven configuration
- Create reusable form layouts and fields
- Use datatables with dynamic configuration
- Implement multistep forms
- Persist UI and form state using Redux
- Use a registry-based system for reusable blocks/components

Developers interact with the system via:
- props
- schemas
- configuration objects
- reusable UI components
